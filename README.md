# storybook-with-figma

storybook と Figma の連携についての調査用リポジトリ

調査結果は`./research.md`に記載

## requires

- Node.js -> v16.13.1

## 依存関係解決

```bash
$ yarn
```

## 開発サーバ起動

```bash
$ yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## storybook 起動

```bash
$ yarn storybook
```
